JAVA SRTEAM 
Java Stream adalah fitur di Java yang digunakan untuk memproses data (terutama data koleksi seperti List, Array, atau Set) secara lebih mudah, cepat, dan ringkas.
Konsep ini diperkenalkan pada Java 8.

Fungsi Java Stream 
1.Memproses data koleksi
2.Menyaring data (filtering),digunakan untuk mengambil data tertentu saja.
3.Mengubah data (mapping),digunakan untuk mengubah isi dalam data.
4.Menghitung jumlah data secara efisien.
5.mengurutkan data 

Keuntungannya 
-kode lebih singkat tapi buat aku bingung heeh
-lebih muda dibaca, sama aja kayak array
-bisa paraller processing
-lebih moder

Perbedaan utama Array dengan Java Stream 
Array                |  Java Stream 
Menyimpan            |  Memproses data
Struktur data        |  API pemrosesan data 
Ukuran tetap         |  Tidak menyimpan data
Diakses dengan index |  Diakses dengan Operasi Stream
